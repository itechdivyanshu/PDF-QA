# PDF-QA
# Setup

1) Installing Required Files For FastAPI

#### `For Windows:`

```lang-python
pip install -r requirements.txt
```
#### `For Linux/Mac:`

```lang-python
pip3 install -r requirements.txt
```

2) Installing Required Files For Node/React

```javascript
cd ./ass-i/
npm install
```
3) Run server:

#### `For FastAPI:`

```lang-python
fastapi dev main.py
```
#### `For React:

```lang-python
cd ./ass-i/
npm start
```
